# Security Policy

## Supported Versions
Security fixes are handled against the latest public eqMacFree branch and the most recent tagged public release.

## Reporting a Security Vulnerability
Please do not open a public issue for undisclosed security reports.

Instead, use one of the following repository-managed channels:

- Open a private GitHub security advisory for this repository.
- Contact the current project maintainer through the security contact documented in the public repo profile.

If those channels are not yet configured at launch time, open a minimal public issue asking for a secure contact path without disclosing exploit details.
